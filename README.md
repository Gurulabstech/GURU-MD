## 🗂 Project Structure

```
GURUH-main/
├── index.cjs           # Main entry point
├── command.js          # Command handler
├── Dockerfile          # Docker config
├── package.json        # Dependencies
├── lib/                # Core utilities & helpers
├── plugins/            # Bot plugins (200+)
│   ├── ai/             # AI chat, image & video tools
│   ├── automation/     # Auto-reply, status, presence
│   ├── anti/           # Anti-link, anti-delete
│   ├── downloader/     # YT, TikTok, FB downloaders
│   ├── fun/            # Games, reactions, Quran
│   ├── group/          # Group management
│   ├── media/          # Image & PDF tools
│   └── social/         # Newsletter, privacy tools
└── config/             # Deployment configs
    ├── heroku.yml
    ├── render.yaml
    ├── koyeb.yaml
    ├── app.json
    └── replit.md
```

## 🚀 Local Setup

```bash
git clone https://github.com/itsguruu/GURU.git
cd GURU
npm install
npm start
```

> **Node.js 20+** required.

Set your `SESSION_ID` environment variable before starting. Get one from the [pairing link](https://pair.gurutech.dev) above.
